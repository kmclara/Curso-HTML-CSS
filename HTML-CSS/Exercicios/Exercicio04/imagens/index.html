Aprendendo sobre: Favicons!

Logo acima você verá a primeira favicon que eu mesma coloquei! 🥰

Para adicionar uma favicon no seu site, devemos fazer o seguinte:

  1. O favicon é dentro da tag head. Então, dentro dela, procuramos pela tag link favicon
  2. Logo depois, é só mudarmos o endereço do href e colocamos o endereço da nossa imagem.

E pronto, nosso favicon é adicionado no nosso site!
