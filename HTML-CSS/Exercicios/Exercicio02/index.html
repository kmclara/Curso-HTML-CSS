Parágrafos e quebras de linha

Podemos escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags p e /p!

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag br

Para que possamos apresentar as tags dentro dos parágrafps com os seus sinais padrão, devemos colocar os seguintes simbolos: <br> , aqui estamos representando a tag br, a tag de quebra de linha.
